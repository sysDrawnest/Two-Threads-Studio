# Two Threads Studio - Visual Design System & Assets

This document outlines the complete visual design system, color palettes, typography, and static image assets for the **Two Threads Studio** e-commerce platform.

---

## 1. Color Palette

The brand uses a warm, sophisticated palette inspired by natural fibers, earth tones, and precious metals.

| Intent               | Name              | Hex Code  | Usage                                                  |
| :------------------- | :---------------- | :-------- | :----------------------------------------------------- |
| **Primary**          | **Rich Charcoal** | `#1C1612` | Premium backgrounds, section separators, primary text. |
| **Primary Light**    | **Soft Taupe**    | `#F9F5EE` | Clean backgrounds, hero sections, card surfaces.       |
| **Accent**           | **Warm Gold**     | `#C9A84C` | CTAs, highlights, active states, borders.              |
| **Secondary Accent** | **Deep Bronze**   | `#A16C39` | Decorative elements, sub-headings.                     |
| **Neutral**          | **Off-White**     | `#FDF9F2` | Body text, subtle borders.                             |

---

## 2. Typography

A serif-heavy editorial system that emphasizes heritage and luxury, contrasted with clean sans-serif for UI clarity.

| Element         | Font Family                  | Usage & Style                                                |
| :-------------- | :--------------------------- | :----------------------------------------------------------- |
| **Heading 1**   | **Playfair Display** (Serif) | "Heritage", "Tradition", "Timeless". High contrast, elegant. |
| **Heading 2**   | **Playfair Display** (Serif) | Section titles (e.g., "Bestsellers", "Curated").             |
| **Heading 3**   | **Playfair Display** (Serif) | Sub-titles, emphasis on collection names.                    |
| **Body Copy**   | **Inter** (Sans-serif)       | Clean, readable text for descriptions, prices, and filters.  |
| **Accent Text** | **Playfair Display** (Serif) | Hand-written style for quotes or specific callouts.          |

## 5. UI Guidelines (Authentication)

* **Glassmorphism Aesthetic**: Auth modals use a glass-like frosted effect (`bg-white/70 backdrop-blur-md border border-white/40 shadow-2xl`) layered over high-resolution texture backgrounds.
* **Immersive Backgrounds**: Use 2K macro shots of embroidery materials (hoops, threads) with subtle contrasting overlays (`contrast(1.1) brightness(0.9) saturate(1.2)` + `mix-blend-overlay`).
* **Interactive Elements**: Floating interactive objects like the "Guest Access Pass" tag, utilizing absolute positioning, SVG connecting strings, and CSS ripple/tilt effects (`hover:-translate-y-1 hover:shadow-xl`).
